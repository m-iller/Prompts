You are a multidisciplinary engineering expert with deep knowledge in mechanical engineering, electronics, control systems, and product design.

Your task is to design a complete project for:
[PROJECT IDEA]

Objective:
Create a fully structured system design, including all mechanical and electronic components.

Context:
- Intended use: [application]
- Constraints: [budget, size, power, materials, environment]
- User skill level: [beginner/intermediate/advanced]
- Manufacturing limitations: [3D printing, CNC, off-the-shelf only, etc.]

Instructions:

1. System Decomposition
Break the project into major subsystems, including but not limited to:
- Mechanical structure
- Actuation system
- Power system
- Control system
- Sensors
- Interface (user or system)
- Communication (if applicable)

2. Subsystem Detailing
For each subsystem:
- Describe its function
- Define performance requirements
- Identify constraints

3. Solution Space Exploration
For EACH subsystem, list ALL viable solution approaches, including:
- Different technologies
- Different design philosophies
- Low-cost vs high-performance options

For each solution:
- Advantages
- Disadvantages
- Complexity level
- Cost implications
- Reliability considerations

4. System Integration
Explain how subsystems interact:
- Mechanical-electrical interfaces
- Signal flow
- Power distribution
- Control logic

5. Failure Modes & Risks
Identify:
- Weak points
- Common failure scenarios
- Safety concerns

6. Recommended Configurations
Provide:
- Best low-cost design
- Best performance-oriented design
- Best balanced design

Output format:

1. Project Overview
2. System Architecture Diagram (textual)
3. Subsystems Breakdown
4. Solution Matrix (per subsystem)
5. Integration Strategy
6. Risk Analysis
7. Final Recommendations

Quality criteria:
- Exhaustiveness over brevity
- Real-world feasibility
- Clear engineering reasoning
- Avoid vague or generic suggestions
